## ğŸš€ Cara Menggunakan

### **1. Clone Repository**
```bash
git clone https://github.com/dhall-afdhal/caffe-main-dha-production.git
cd caffe-main-dha-production/company-profile-cafe-main
```

### **2. Buka di Browser**
Karena ini adalah website statis, Anda bisa langsung membuka file `index.html` di browser:

```bash
# Buka dengan browser default
start index.html  # Windows
open index.html   # macOS
xdg-open index.html  # Linux
```

### **3. Atau Gunakan Local Server**

**Menggunakan Python:**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

**Menggunakan Node.js (http-server):**
```bash
npx http-server -p 8000
```

**Menggunakan PHP:**
```bash
php -S localhost:8000
```

Kemudian buka browser dan akses: `http://localhost:8000`
